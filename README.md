# 🧠 LeetCode Solutions & Algorithmic Thinking

This repository contains my personal solutions to LeetCode problems, written in **Python 3**. 

The main goal of this space is to transition from brute-force approaches to highly optimized solutions, focusing heavily on minimizing both **Time Complexity** and **Space Complexity** in preparation for computational challenges.

# 🎯 Key Techniques Overview

-**Two Pointers Technique:** Optimizing array manipulation and sliding windows in $\mathcal{O}(N)$ time complexity and $\mathcal{O}(1)$ space (e.g., Remove Duplicates, Palindrome Number).

-**Hash Maps & Dictionaries:** Reducing lookup times to $\mathcal{O}(1)$ for element mapping (e.g., Two Sum).

-**Stack Data Structure:** Managing nested linear patterns and tracking elements chronologically (e.g., Valid Parentheses).

-**Binary Search:** Tackling sorted datasets to achieve logarithmic $\mathcal{O}(\log N)$ time efficiency (e.g., Median of Two Sorted Arrays).
